# Daffodil International University - Student Portal

A modern ReactJS front-end application built with Vite and TailwindCSS for Daffodil International University students. 

## Features
- Student Dashboard & Profile
- Course Registration & Grades 
- Class Schedule & Routine
- Library Services & Payments Ledger
- Announcements & Notices

## Latest Updates (Main Branch)
- Core layout and routing completed
- Auth pages integration (Login, Signup)
- Dashboard overview added

## How to Run

1. **Install Dependencies:**
   ```bash
   npm install
   ```
2. **Start the Development Server:**
   ```bash
   npm run dev
   ```
3. **Open in Browser:**
   Navigate to `http://localhost:5173`

## Built With
- ReactJS (Vite)
- TailwindCSS
- React Router DOM
- React Icons
