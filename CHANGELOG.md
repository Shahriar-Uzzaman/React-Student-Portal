# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-04-23

### Added
- Initial ReactJS project structure using Vite.
- Core infrastructure: React Router, TailwindCSS, and layout components.
- Authentication system: Login and Signup pages with DIU branding.
- Student Dashboard: Overview with stats, grades, and notifications.
- Student Portal Pages: Profile, Courses, Schedule, Grades, and Settings.
- Mock DIU student data integration.
- Dark mode support via ThemeContext.

### Fixed
- Resolved merge conflicts during feature integration.
- Demonstrated and verified Git undo/recovery workflows (revert/reflog).
